# Cloud Inventory Manager

## Description

Cloud Inventory Manager is a Python application that uses Google Firebase Firestore as a cloud database. The program allows users to add, view, update, and delete inventory products through a console-based menu system.

This project was created for Module 3 of CSE 310 Applied Programming to demonstrate the use of cloud databases and CRUD operations.

## Features

* Add products to the cloud database
* View all stored products
* Update existing products
* Delete products from the database
* Store inventory information remotely using Firebase Firestore

## Technologies Used

* Python
* Firebase Firestore
* Firebase Admin SDK

## How to Run the Program

1. Install the required packages:

pip install -r requirements.txt

2. Place the Firebase service account key file in the project folder.

3. Run the application:

python main.py
